Overview

This repository is a lightweight Next.js app (App Router) starter template built with TypeScript and Tailwind CSS. It demonstrates a minimal application layout using the Next.js app directory, custom Google fonts via next/font, and a simple example page that uses the Next.js Image component and links.

Primary purpose

- Provide a minimal, opinionated starter for Next.js projects using the App Router and Tailwind CSS.
- Serve as a local template that can be extended to build applications.

How to run locally

1. Install dependencies

   npm install

2. Run the development server

   npm run dev

3. Build for production

   npm run build

4. Start the production server

   npm run start

Observations & recommendations

- The template is intentionally minimal; expand the app directory with additional routes, components, and layouts to fit your project structure.
- Tailwind is already configured — update tailwind config if you need custom themes or additional plugins.
- The project uses next/font/google for font loading. If you need self-hosted fonts, move them into public/ and update layout.tsx accordingly.
- Ensure TypeScript strictness settings are adjusted in tsconfig.json per your project's needs.
